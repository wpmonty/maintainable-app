# maintainable.app

Email-based habit tracker powered by AI. No app, no account, no streaks — just email what you did today.

## Repo Structure

```
maintainable-app/
├── site/              # Marketing site (React + Vite + Tailwind)
├── MARKETING.md       # Full marketing plan (segments, GTM, pricing)
├── PLANS.md           # Technical architecture + roadmap
└── README.md          # This file
```

## Quick Start

```bash
# Marketing site
cd site && npm install && npm run dev
# Opens on http://localhost:9800
```
